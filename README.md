# AndroidhotoEditor

## Overview
AndroidPhotoEditor s a simple photo editor for Android. AndroidPhotoEditor allows users to cropping, drawing and typing. 

## Features
- [x] Cropping
- [x] Drawing with colors
- [x] Typing with colors
